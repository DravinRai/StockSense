# StockSense Mobile App

The mobile frontend for the StockSense Investment Companion, built with Expo and React Native.

## 🚀 Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment:
   Create a `.env` file in this directory:
   ```bash
   EXPO_PUBLIC_API_URL=http://your-local-ip:3001
   ```

3. Start the app:
   ```bash
   npx expo start
   ```

## ✨ Highlights
- **Groww-style Dashboard**
- **AI Stock Insights**
- **Interactive Charts** (Wagmi, Victory)
- **Portfolio Tracking**
